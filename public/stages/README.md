# Stage Background Images

This directory contains background images for battle stages.

## Required Images

Add the following images to this directory:

- `canada.jpg` - Background image for "Futur's Den" (Canada stage)
- `bronx.jpg` - Background image for "1520 Sedgwick Avenue" (Bronx, NY - birthplace of hip-hop)

## Image Guidelines

- **Recommended dimensions:** 1920x1080 or higher (16:9 aspect ratio)
- **Format:** JPG or PNG
- **File size:** Keep under 500KB for optimal web performance
- **Style:** High-quality images that represent the location/vibe of each stage

## How to Add Images

1. Find or create images for each stage
2. Optimize them for web (compress if needed)
3. Name them exactly as listed above
4. Place them in this `/public/stages/` directory
